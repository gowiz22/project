# project-PETTI
스프링 부트 + JSP 반려 동물 커뮤니티

## 프로젝트 소개
반려동물에 대한 지식과 정보를 얻고 반려동물 제품 리뷰를 확인할 수 있는 커뮤니티 사이트입니다.

* 'JDK 11.0.15'
* IDE : STS 3.9
* Spring Framework : web mvc(5.2.24)
* Database : Oracle DB(11g xe)
